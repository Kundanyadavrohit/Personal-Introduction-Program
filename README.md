# 🚀 Personal Introduction Program

A simple Python console application that collects a user's personal information and creates a basic profile introduction.

## 📌 Features

- Welcomes the user with a formatted interface.
- Takes user input for:
  - Name
  - Age
  - Favorite Hobby
- Validates that the inputs are not empty.
- Displays a personalized introduction message.
- Beginner-friendly Python project for learning functions, loops, input handling, and string formatting.
